參考題目連結連結

1.

題目:

if(false){
var a = 1;
let b = 2;
}
console.log(a);
console.log(b);

輸出:
a undefined未明確定義;b is not defined

解析:
因為if層級高，a進不去=1，但是var a 屬於全域變數，而let b 為區域變數，not defined。

2.

題目:

var a = 1;
if(true){
console.log(a);
let a =2;
}

輸出:
Cannot access 'a' before initialization

解析:
因為if 提升比var高，先執行，到console.log(a)=>出現還沒執行這一段之前你不能使用這個變數

3.

題目:

var a = {n:1};
var b = a;
a.x = a = {n:2};
console.log(a.n,b.n);
console.log(a.x,b.x);

輸出:
2 1
undefined object{n:2}

解析:
var a.x =>a擁有一個屬性x;先執行;
a =物件{n:1}
b = a =物件{n:1}且 帶有 a.x =擁有x屬性;
a =物件{n:2}(賦值由右至左，創建一個新的a)
a.x= a 物件{n:2}
最後
a.n=2;
b.n=1;
a.x=undefined;
b.x=物件{n:2};
